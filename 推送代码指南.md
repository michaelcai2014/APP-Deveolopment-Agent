# 推送代码到 GitHub 指南

## 方式 1：使用 Personal Access Token（推荐）

### 步骤 1：创建 Token
1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token" → "Generate new token (classic)"
3. 设置名称（如 "App Development Agent"）
4. 勾选 `repo` 权限
5. 点击 "Generate token"
6. **复制生成的 token**（只显示一次，请保存好）

### 步骤 2：推送代码
在终端执行以下命令：

```bash
cd /Users/tunda/Desktop/project/app-mvp-estimator

# 使用 HTTPS 方式
git remote set-url origin https://github.com/michaelcai2014/APP-Deveolopment-Agent.git

# 推送代码（会提示输入用户名和密码）
git push -u origin main
```

**输入提示时：**
- Username: 输入你的 GitHub 用户名 `michaelcai2014`
- Password: 输入刚才复制的 **Personal Access Token**（不是 GitHub 密码）

## 方式 3：配置 SSH 密钥（长期方案）

### 步骤 1：生成 SSH 密钥
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
# 按回车使用默认路径
# 设置密码（可选）
```

### 步骤 2：添加 SSH 密钥到 GitHub
```bash
# 复制公钥内容
cat ~/.ssh/id_ed25519.pub
```

然后：
1. 访问：https://github.com/settings/keys
2. 点击 "New SSH key"
3. 粘贴公钥内容
4. 点击 "Add SSH key"

### 步骤 3：推送代码
```bash
cd /Users/tunda/Desktop/project/app-mvp-estimator
git remote set-url origin git@github.com:michaelcai2014/APP-Deveolopment-Agent.git
git push -u origin main
```
